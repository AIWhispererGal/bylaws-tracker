## Color Palette

```
Primary Gradient: #667eea → #764ba2 (Purple)
Success: #10b981 (Green)
Danger: #ef4444 (Red)
Info: #3b82f6 (Blue)
Warning: #f59e0b (Orange)

Text:
  Primary: #2d3748 (Dark Gray)
  Secondary: #6b7280 (Medium Gray)
  Muted: #9ca3af (Light Gray)

Backgrounds:
  Primary: #ffffff (White)
  Secondary: #f9fafb (Very Light Gray)
  Tertiary: #f3f4f6 (Light Gray)

Borders:
  Light: #e5e7eb
  Medium: #d1d5db
  Dark: #9ca3af
```

## Animations

1. **Page transitions**: Fade in (0.3s)
2. **Progress pulse**: Scale 1.0 → 1.05 → 1.0 (2s loop)
3. **Card hover**: Translate Y -8px + shadow (0.3s)
4. **Confetti**: Fall from top with rotation (3s)
5. **Checkmark**: Scale in with bounce (0.5s)
6. **Loading spinner**: Rotate 360° (1s loop)

---

**Note**: These are ASCII representations. The actual implementation uses HTML, CSS, and JavaScript for a polished, professional appearance.
